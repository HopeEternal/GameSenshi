# Game Senshi

A platform for gamer to provide their casual service.

## Getting Started

A step by step series of examples that tell you how to get a development env running.

first:
```
npm i
```

then:  
request the `.config` folder from `tylim#9831` in `Discord`

finally:
```
npm run dev
```
